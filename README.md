# CHIP8 <br>
CHIP8 emulator written in rust <br>
**Dependencies:** <br>
SDL2 installed on your machine <br>
**Keyboard Layout:** <br>
**Chip8 Keypad:** <br>
1	2	3	C <br>
4	5	6	D <br>
7	8	9	E <br>
A	0	B	F <br>
**Emulator Keyboard Mapping:** <br>
1	2	3	4 <br>
Q	W	E	R <br>
A	S	D	F <br>
Z	X	C	V <br>
'esc' Key : Close the Emulator <br>
