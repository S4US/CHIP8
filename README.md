# CHIP8
CHIP8 emulator written in rust
**Dependencies:**
SDL2 installed on your machine
**Keyboard Layout:**
**Chip8 Keypad:**
1	2	3	C\n
4	5	6	D
7	8	9	E
A	0	B	F
**Emulator Keyboard Mapping:**
1	2	3	4
Q	W	E	R
A	S	D	F
Z	X	C	V
'esc' Key : Close the Emulator
'Spacebar' : Pause / Resume the Emulator
'F5 Key' : Reset the emulator
